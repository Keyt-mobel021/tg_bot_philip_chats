# Filter Bot — Бот фильтрации сообщений в чатах

Бот для автоматической фильтрации сообщений в Telegram группах.
Удаляет сообщения, содержащие запрещённые паттерны (regex + нечёткое совпадение).

## Структура проекта

```
filter_bot/
├── app.py                  # Точка входа
├── config.py               # Конфигурация
├── models.py               # Модели базы данных (SQLite)
├── states.py               # FSM состояния
├── filters.py              # Фильтр CheckAdmin
├── requirements.txt
├── .env.example
├── handlers/
│   ├── admin.py            # Админ-панель (личный чат)
│   └── group.py            # Фильтрация сообщений в группах
└── keyboards/
    ├── __init__.py         # CallbackData классы
    └── kb.py               # Клавиатуры
```

## Установка

```bash
pip install -r requirements.txt

# Заполните .env своими данными

```

## Как работает

### Для администраторов (личный чат с ботом)
1. `/start` → меню с кнопкой **Чаты**
2. Выбираете чат → видите информацию + кнопки управления
3. **Создать фильтр** — вводите regex выражение
4. **Редактировать фильтр** — выбираете из списка → вводите новое выражение
5. **Удалить фильтр** — выбираете из списка → удаляется
6. **Порог срабатывания** — процент нечёткого совпадения (0-100, рек. 75-85)

### Для групп
1. Добавляете бота в группу как администратора
2. Пишете `/start` в группе → группа регистрируется
3. Бот автоматически удаляет сообщения, подпадающие под фильтры

### Алгоритм фильтрации
1. **Regex проверка**: сообщение проверяется по регулярному выражению
2. **Нечёткое совпадение (fuzzywuzzy)**: слова из текста сравниваются с ключевыми словами паттерна. Если совпадение >= порога — сообщение удаляется

### Примеры regex паттернов
- `\+7\d{10}` — номера телефонов (+7XXXXXXXXXX)
- `8\d{10}` — номера телефонов (8XXXXXXXXXX)  
- `@\w+` — упоминания в Telegram
- `https?://\S+` — любые ссылки
- `(?i)(телефон|звони|созвонимся|сотовый)` — ключевые слова

Тестировать regex удобно на: https://regex101.com/
